## Project
### Password Generator 

Built both the **Easy** and **Hard** versions of a password generator.

* User chooses:

  * Number of letters
  * Number of symbols
  * Number of numbers
* Generates a random password.
* Hard version shuffles characters for better randomness.

I also practiced combining loops, lists, and the `random` module to build both an Easy and a Hard Password Generator. This project improved my logical thinking and helped me understand how multiple Python concepts work together.
