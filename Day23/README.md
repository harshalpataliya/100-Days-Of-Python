# Day 23 - Turtle Crossing Game

A simple arcade-style **Turtle Crossing Game** built using Python's `turtle` graphics module as part of **Day 23** of the **100 Days of Code: The Complete Python Pro Bootcamp** by Dr. Angela Yu.

## Project Overview

In this game, the player controls a turtle that must safely cross a busy road filled with moving cars. Each successful crossing increases the level and makes the cars move faster, increasing the game's difficulty.

The project demonstrates how multiple classes work together to create an interactive game using Object-Oriented Programming (OOP).


- Player-controlled turtle
- Randomly generated moving cars
- Collision detection
- Level progression
- Cars speed up after every successful crossing
- Game Over screen
- Modular object-oriented design

##  Learned

- Object-Oriented Programming (OOP)
- Creating and managing multiple objects
- Working with lists of objects
- Random object generation
- Collision detection using `distance()`
- Keyboard event handling
- Game loops
- Screen animation with `screen.update()`
- Increasing game difficulty
- Organizing projects into multiple Python files


## Learning Outcome

This project helped me understand how to manage multiple moving objects using lists, detect collisions between game objects, organize code using multiple classes, and create a complete game with increasing difficulty.


Gameplay includes:

- Turtle controlled by the **Up Arrow** key
- Randomly generated traffic
- Level increases after each successful crossing
- Faster cars every level
- Game Over on collision
