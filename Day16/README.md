# Day 16 - Coffee Machine using Object-Oriented Programming (OOP)

## Overview

Today I rebuilt the Coffee Machine project using Object-Oriented Programming (OOP). Instead of writing all the functions from scratch, I learned how to use classes and objects to build a complete application. I also learned how Python packages are imported and used in a project.

## What I Learned

- Python packages and modules
- Importing classes from different Python files
- Installing Python packages using `pip`
- Object-Oriented Programming (OOP)
- Creating objects from classes
- Calling methods using objects
- Understanding the relationship between multiple classes
- Using the Turtle module
- Organizing code into separate files
- Building applications using reusable code

## Project Features

- Display available coffee options
- Generate resource and money reports
- Check if enough resources are available
- Process customer payments
- Return change when required
- Prepare coffee after successful payment
- Turn off the machine using the `off` command

## Files

- `main.py` - Main program
- `menu.py` - Menu and drink information
- `coffee_maker.py` - Coffee machine resource management
- `money_machine.py` - Payment processing

## OOP Concepts Practiced

- Classes
- Objects
- Constructors (`__init__`)
- Methods
- Object Interaction
- Encapsulation

## Modules Used

- `menu`
- `coffee_maker`
- `money_machine`
