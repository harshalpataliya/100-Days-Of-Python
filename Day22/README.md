# Day 22 - Pong Game 

## Project Overview


This project focuses on Object-Oriented Programming by creating separate classes for the paddles and ball while implementing smooth movement, collision detection, and game animation.

## What I Learned

- Creating custom classes using Object-Oriented Programming (OOP)
- Working with multiple Python files
- Inheriting from the `Turtle` class
- Using constructors (`__init__`)
- Creating reusable objects with parameters
- Handling keyboard events with `screen.onkey()`
- Using `screen.tracer(0)` and `screen.update()` for smooth animation
- Moving objects continuously inside a game loop
- Detecting collisions with screen boundaries
- Making the ball bounce by changing its movement direction
- Managing object positions using `xcor()`, `ycor()`, and `goto()`

##  Technologies Used

- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)

## Features

- Two-player Pong game
- Smooth paddle movement
- Ball movement and wall bounce
- Keyboard controls
- Real-time game animation
