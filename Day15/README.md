# Day 15 - Coffee Machine

## Overview

Today I built a Coffee Machine program in Python. The program simulates a coffee machine that can serve espresso, latte, and cappuccino. It checks available resources, processes coin payments, gives change, updates ingredients, and keeps running until the machine is turned off.

## What I Learned

- Using dictionaries to store structured data
- Creating and using multiple functions
- Returning values from functions
- Working with global variables
- Using loops to keep a program running
- Checking conditions with if/elif/else
- Managing program flow
- Updating dictionary values

- Order espresso, latte, or cappuccino
- Display machine report
- Check if enough resources are available
- Process coin payments
- Refund money if payment is insufficient
- Return change when needed
- Update machine resources after each order
- Turn off the machine using the secret command `off`

- Functions
- Dictionaries
- Loops
- Conditionals
- Global Variables
- Return Statements
