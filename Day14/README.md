## Project

Built a Higher Lower game in Python. The game randomly selects two famous people and asks the player to guess who has more followers. The game continues until the player makes a wrong guess.

## Files

- `main.py` - Main game logic
- `game_data.py` - Contains the data used in the game
- `art.py` - Stores the game logo and visual elements

## What I Learned

- How to use `random.choice()` to select random data.
- How to access values from dictionaries.
- How to return a dictionary from a function.
- How to compare values using conditional statements.
- How to organize code using functions.
- How to build a complete Python game.
