## Project

Built a Higher Lower game in Python. The game randomly selects two famous people and asks the player to guess who has more followers. The game continues until the player makes the wrong guess and displays the final score.

## Files

- `main.py` - Main game logic
- `game_data.py` - Contains the data used in the game
- `art.py` - Contains the game logo and visual elements

## What I Learned
