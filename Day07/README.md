##  Project

Built the first version of the classic Hangman game.

The project includes:

* Random word selection
* Repeated user guesses
* Letter matching
* Hidden word display using underscores (`_`)
* Revealing correctly guessed letters in their correct positions
* Continuing the game until the entire word is guessed


## Files

* `main.py` – Hangman game .
* `hangman_art.py` - ASCII.arts.
* `hangman_word.py` - Contain a list of words.
