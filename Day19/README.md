# Day 19 -

## Project
A simple Turtle Race game built using Python's `turtle` module. The user places a bet on which turtle will win, and six turtles race across the screen with random movements until one reaches the finish line.

## Concepts Learned

- Higher Order Functions
- Event Listeners
- `listen()` Method
- `onkey()` and `onkeypress()`
- Objects, State, and Instances
- Lists of Objects
- Loops
- Random Module
- Turtle Graphics

- User enters a turtle color as a bet.
- Six turtles start from different positions.
- Each turtle moves a random distance every turn.
- The race ends when a turtle crosses the finish line.
- The program announces whether the user's prediction was correct.

## Technologies Used
- Python 3
- Turtle Module
- Random Module
