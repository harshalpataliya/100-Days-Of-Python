# Day 20 & 21 - Snake Game 

## Overview
In Day 20 and Day 21 of the **100 Days of Code: The Complete Python Pro Bootcamp** by Dr. Angela Yu, I built the classic **Snake Game** using Python's Turtle graphics module while learning Object-Oriented Programming (OOP) concepts.

## What I Learned
### Object-Oriented Programming (OOP)
- Creating and using custom classes
- Understanding objects and instance attributes
- Using the `__init__()` constructor
- Working with `self`
- Calling object methods
- Organizing code into multiple Python files

### Inheritance
- Inheriting from the `Turtle` class
- Using `super().__init__()`
- Creating custom `Food` and `Scoreboard` classes

### Turtle Graphics
- Moving the snake using Turtle objects
- Controlling movement with keyboard events
- Detecting collisions using `distance()`
- Getting object coordinates with `xcor()` and `ycor()`
- Displaying and updating text with `write()` and `clear()`

### Python Concepts
- Lists and list slicing
- Loops and conditional statements
- Working with object attributes and methods
- Game loop using `while`
- Collision detection
- Code organization and debugging

## Features
- Snake movement using arrow keys
- Food spawning at random locations
- Snake grows after eating food
- Live score display
- Wall collision detection
- Self-collision detection
- Game Over screen

## Technologies Used
- Python 3
- Turtle Graphics
- Object-Oriented Programming (OOP)
